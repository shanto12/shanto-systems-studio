# Design Brief

## Product Premise

Shanto Systems Studio is a premium motion website that presents Shanto's agentic AI, security automation, XSIAM/SOAR, demo-factory, and live-ops work as one proof-driven system.

## First 30 Seconds

The viewer should immediately see a high-end motion site with a day/night hero transition, strong typography, real controls, and a credible security automation story.

## Visual Direction

- Pure black base with graphite glass surfaces.
- Electric cyan, signal green, amber, and small coral accents.
- Cinematic mountain operations landscape for the hero.
- Shanto Mathew's LinkedIn profile photo used as a local operator signal.
- Code-native text and controls.
- Motion supports hierarchy: no decorative-only UI that cannot be clicked.

## Required Sections

1. Hero with paired day/night transition.
2. Section cards previewing the rest of the site.
3. Observe / Plan / Act / Verify workflow.
4. Interactive systems map.
5. Proof timeline.
6. Motion lab.
7. Demo guide and release checklist.
